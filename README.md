# \# Tanks Battle Arena

# 

# Проєкт створено на основі навчального туторіалу Unity Tanks: Make a battle game for web and mobile.

# 

# \## Реалізовано за туторіалом

# \- керування танком;

# \- стрільба снарядами;

# \- система здоров’я;

# \- вибухи та урон;

# \- ігрова камера;

# \- Game Manager;

# \- умови перемоги та поразки.

# 

# \## Власні покращення

# \- розширено ігрову арену;

# \- додано більше перешкод та укриттів;

# \- змінено візуальне оформлення рівня;

# \- додано звукові ефекти;

# \- додано particle effects для вибухів та підбирання бонусів.

# 

# \## Додаткові механіки

# 1\. Speed Boost - бонус тимчасово збільшує швидкість танка.

# 2\. Shield Bonus - бонус тимчасово захищає танк від урону.

# 3\. Slow Trap - пастка тимчасово зменшує швидкість танка.

# 

# \## Керування

# \- WASD / Arrow Keys - рух танка.

# \- Space - постріл.

# 

# \## Авторські зміни

# У проєкті змінено ігрову сцену, додано нові механіки, бонуси, пастки, звукові ефекти та покращено оточення.
