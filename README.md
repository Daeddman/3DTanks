# Tanks Battle Arena

Проєкт створено на основі навчального туторіалу Unity Tanks: Make a battle game for web and mobile.

## Реалізовано за туторіалом
- керування танком;
- стрільба снарядами;
- система здоров’я;
- вибухи та урон;
- ігрова камера;
- Game Manager;
- умови перемоги та поразки.

## Власні покращення
- розширено ігрову арену;
- додано більше перешкод та укриттів;
- змінено візуальне оформлення рівня;
- додано звукові ефекти;
- додано particle effects для вибухів та підбирання бонусів.

## Керування
- WASD / Arrow Keys - рух танка.
- Space - постріл.
